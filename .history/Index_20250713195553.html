Pyetja 1

Si të ngarkosh një projekt në GitHub nga Visual Studio Code

Hapat Kryesorë

  • 1. Krijo një repository të ri në GitHub (publik)
  • 2. Hap projektin në Visual Studio Code
  • 3. Inicializo Git në projektin tënd
  • 4. Lidhu me repository-n në GitHub
  • 5. Bëj push për të ngarkuar projektin

1. Krijo një repository të ri në GitHub

Shko te https://github.com, kyçu në llogarinë tënde dhe kliko në butonin "New repository". Jep emrin e projektit, përshkrimin dhe zgjidh opsionin "Public". Mos zgjedh krijimin e README nëse ke tashmë projektin në kompjuter.

2. Hap projektin në Visual Studio Code

Hap Visual Studio Code dhe nga menyja zgjidh File > Open Folder dhe zgjidh dosjen e projektit tënd që do të ngarkosh në GitHub.

3. Inicializo Git në projekt

Hap terminalin dhe shkruaj komandat në vijim:

git init
git add .
git commit -m "Commit i parë"

4. Lidhu me repository-n në GitHub

Kopjo URL-në e repository-t nga GitHub dhe lidhe atë me projektin lokal me komandën:

git remote add origin https://github.com/emri-yt/emri-projektit.git

5. Ngarko projektin në GitHub

Shkruaj komandën për të ngarkuar projektin në GitHub:

git push -u origin master

Ose nëse dega e paracaktuar është main:

git push -u origin main
